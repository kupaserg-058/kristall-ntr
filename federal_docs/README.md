# Федеральные документы для результата 1

Назначение папки: локально собрать федеральную рамку, на которую опираются пилотный набор, промт извлечения направлений НТР, эталонная разметка и расчет качества MVP.

Главный документ в этой папке - `Указ 529.pdf`. Он задает классификатор проекта: 7 направлений НТР и перечень важнейших наукоемких технологий.

## Как использовать в разработке

Для первой версии MVP не нужно загружать все федеральные документы в обработку как равные источники. Практически достаточно:

1. Закодировать классификатор из F2: 7 направлений и ключевые технологические маркеры.
2. Добавить понятия из F1, F3, F4 и F5 как правила интерпретации.
3. Использовать F11-F13 только как расширение словаря отраслевых признаков.
4. Отдельно помечать F7 и F8 как источники с неполной текстовой доступностью.

## Что нельзя делать

- Нельзя считать наличие слова "инновации" автоматическим направлением НТР.
- Нельзя оценивать регион выше только за прямую ссылку на федеральный акт, если нет технологий, проектов, организаций и KPI.
- Нельзя использовать рейтинг НТР как доказательство качества конкретного регионального документа.
- Нельзя без проверки делать выводы по F7 и F8, если используется только неполный OCR.
